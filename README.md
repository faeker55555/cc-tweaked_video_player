# Custom Video Player with Subtitle Support

A specialized video player implementation featuring enhanced closed caption (CC) rendering and synchronization controls.

## Features

- Custom subtitle/caption overlay system
- Pause/resume functionality with frame-accurate positioning
- Cross-platform video playback support

## Demo Content

This repository includes a sample video file (*Interstellar*, Russian audio) for demonstration purposes.

**Note on Audio Synchronization:** Under certain playback conditions, audio drift may occur. Pausing and resuming playback typically re-synchronizes the audio stream.
